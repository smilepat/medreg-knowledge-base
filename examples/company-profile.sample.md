# 회사 역량 프로필 (예시·가상 — bid_matrix 충족판정 시연용)

> 실제 사용 시 이 양식을 `examples/company-profile.md`(또는 별도 경로)로 복사해
> 자사 정보로 채우세요. 실제 회사 자료는 .gitignore로 git 제외됩니다.

## 인허가·자격
- 의료기기 **제조업 허가** 보유 (식품의약품안전처)
- 의료기기 **수입업 허가** 미보유

## 품질·인증
- **GMP 적합인정서** 보유 (의료기기 제조 및 품질관리기준, 유효기간 내)
- ISO 13485 인증 보유

## 제품 (품목허가/인증 현황)
- 초음파 수술기: **품목허가** 완료 (2등급)
- 의약품 주입펌프: **품목허가** 완료 (2등급)
- 의료용 주사기: 품목신고 완료 (1등급)

## 보험 등재
- 일부 주입펌프 부속 치료재료: **국민건강보험 요양급여 등재** 완료
- 초음파 수술기: 비급여(등재 없음)

## 납품 실적
- 최근 3년간 공공의료기관(국립대병원 등) 납품 실적 **다수 보유**(증빙 가능)

## 비고
- 입찰가격은 건별로 산정(프로필에 고정값 없음)
